

# Vue-Learning-Project

### Vue框架学习实操项目

### 上手指南

###### 开发前的配置要求

1. Nodejs
2. Vue
3. Mock

###### **安装步骤**

```sh
git clone https://github.com/Son4ta/Vue-Learning-Project.git
```



### 部署

###### **项目部署**

```sh
npm install
```

###### **开发时热加载编译**

```python
npm run serve
```

###### **生产环境下最小体积编译**

```
npm run build
```

###### **检查和修复文件**

```
npm run lint
```
